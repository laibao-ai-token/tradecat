# 验证状态（最小可执行）

## 运行方式

使用 5 秒超时做“启动级”冒烟检查，避免长任务阻塞：

```
timeout 5s python3 <impl>.py
```

日志输出保存在：

```
services-preview/datacat-service/logs/smoke-*.log
```
